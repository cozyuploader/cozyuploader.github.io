자주 묻는 질문

YouTube 계정이 필요한가요?

네. 업로드할 YouTube 채널의 Google 계정이 필요합니다. OAuth 2.0 으로 인증하며 본인 채널에만 접근합니다.

API 비용은 어떻게 되나요?

Gemini API · YouTube API 모두 사용자 본인 계정 키를 사용하며, 일반 개인 채널 운영에서는 무료 한도 안에서 가능합니다 (신용카드 등록 불필요). 가이드: YouTube API 등록 가이드 27단계

등급별 채널 / 업로드 한도는?

💡 1채널 — 채널 1개 · 1일 1영상 (1년 365영상)
🌱 2채널 — 채널 2개 · 1일 2영상 (1년 730영상)
🚀 3채널 — 채널 3개 · 1일 3영상 (1년 1,095영상)
👑 6채널 — 채널 6개 · 1일 6영상 (1년 2,190영상)
전 등급 Google 계정 1개 · 채널당 1편/일 · 완전 자동화 포함.

완전 자동화는 어느 등급부터 되나요?

1채널 라이선스부터 전 등급에서 완전 자동화가 포함됩니다. 매일 자정 00:00 에 채널당 1편씩 자동 생성·예약 업로드하고, 새벽 03:00 에 어제 성과를 분석해 다음 기획에 반영합니다.

계정(아이디)은 몇 개까지 되나요?

전 등급 Google 계정 1개 기준입니다. 한 계정 아래에서 등급에 따라 1~6개 채널을 한 화면에서 통합 관리합니다. 채널을 클릭하면 그 채널의 모든 설정이 펼쳐집니다.

왜 "채널당 1편/일" 인가요?

YouTube 알고리즘은 일정한 업로드 주기를 선호합니다. 채널당 1편/일 원칙을 지켜 안정적 노출과 구독자 신뢰를 확보합니다. 상위 등급은 다채널로 일일 총량을 늘립니다.

할인 쿠폰은 어떻게 쓰나요?

사이트에 표시된 금액은 정가입니다. 발급받으신 할인 쿠폰을 결제 단계에서 입력하면 쿠폰에 적용된 할인가로 구매하실 수 있습니다.

1년 라이선스가 끝나면?

1년 라이선스 만료 후에는 월 구독료(1채널 99,000원 / 2채널 179,000원 / 3채널 249,000원 / 6채널 449,000원)로 계속 운영하거나, 1년 단위로 갱신하거나, 평생사용권으로 업그레이드할 수 있습니다. 월 구독은 처음부터 구매하는 상품이 아니라 1년 라이선스 사용자를 위한 연장 옵션입니다.

1년 라이선스와 평생 라이선스는 뭐가 다른가요?

1년 라이선스는 한 번 결제로 1년간 사용하고, 만료 후 월 구독 · 1년 갱신 · 평생 업그레이드 중에서 선택합니다. 평생 라이선스는 한 번 결제로 추가 비용 없이 평생 사용하며 갱신이 필요 없습니다. 두 라이선스의 기능은 동일하고, 사용 기간과 결제 방식만 다릅니다.

자동화 도중 다른 계정으로 로그인하면?

로그아웃하는 즉시 자동화는 완전히 정지되며, 다른 사용자 데이터에 접근하지 않습니다. 사용자별 데이터는 모두 격리 저장됩니다.

AI 결과물(이미지/영상)의 저작권은?

Google Gemini API 이용약관에 따라 사용자(키 발급자)에게 있습니다. 다만 결과물이 기존 저작물과 유사할 수 있어 업로드 전 확인을 권장합니다.

업로드 음원의 저작권은?

본인 제작 또는 정당하게 라이선스 확보한 음원만 업로드 가능. 저작권 침해 책임은 업로드한 사용자에게 있습니다.

해상도는 어떻게 되나요?

디폴트는 1080p Full HD (1920×1080) 이며, 전 등급에서 1440p (2K) 또는 4K UHD 도 선택 가능합니다. CPU 기반 libx264 인코딩, GPU 가속(NVENC/AMF)은 향후 업데이트 예정.

MacOS / Linux 도 되나요?

현재 Windows 10/11 64bit 전용입니다. macOS / Linux / 32bit Windows / 모바일은 미지원. 사양 미충족 환경에서 발생하는 동작 불량은 환불 사유로 인정되지 않습니다.

환불 정책은?

공식 판매 채널의 이용약관을 따릅니다. 라이선스 미활성 시 14일 이내 전액 환불 가능. 자세한 사항은 이용약관 참고.
