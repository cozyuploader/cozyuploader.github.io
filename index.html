자주 묻는 질문

YouTube 계정이 필요한가요?

네. 업로드할 YouTube 채널의 Google 계정이 필요합니다. OAuth 2.0 으로 인증하며 본인 채널에만 접근합니다.

API 비용은 어떻게 되나요?

Gemini API · YouTube API 모두 사용자 본인 계정 키를 사용하며, 일반 개인 채널 운영에서는 무료 한도 안에서 가능합니다 (신용카드 등록 불필요). 가이드: YouTube API 등록 가이드 27단계

등급별 채널 / 업로드 한도는?

💡 베이직 — 1 채널 · 1일 1영상 (1년 365영상)
🚀 프로 — 6 채널 · 1일 6영상 (1년 2,190영상), 채널별 독립 설정
👑 프리미엄 — 18 채널 · 1일 18영상 (1년 6,570영상), 아이디 3개 자동 전환

왜 "채널당 1편/일" 인가요?

YouTube 알고리즘은 일정한 업로드 주기를 선호합니다. 채널당 1편/일 원칙을 지켜 안정적 노출과 구독자 신뢰를 확보합니다. 프로/프리미엄은 다채널로 일일 총량을 늘립니다.

1년 라이선스가 끝나면?

1년 라이선스 만료 후에는 월 구독료(베이직 49,000원 / 프로 198,000원 / 프리미엄 450,000원)로 계속 사용하거나, 평생사용권으로 업그레이드 가능합니다. 펀딩 구매자는 평생 펀딩 가격 보장.

정식 출시 후 가격 인상은 왜?

Cozy Uploader 는 현재 와디즈 한정 판매 중입니다. 정식 출시 시점에 다른 판매 채널이 오픈되면 광고비·운영비·고객 지원 인력 비용으로 1년/평생권 ×3, 월 구독료 ×2 인상됩니다. 펀딩 구매자에게는 영향 없습니다.

베이직은 정말 1채널만 되나요?

네. 보안상 한 번 채널을 등록하면 다른 채널로 변경할 수 없습니다. 변경이 필요하면 고객센터 문의로 처리됩니다.

자동화 도중 다른 계정으로 로그인하면?

로그아웃하는 즉시 자동화는 완전히 정지되며, 다른 사용자 데이터에 접근하지 않습니다. 사용자별 데이터는 모두 격리 저장됩니다.

AI 결과물(이미지/영상)의 저작권은?

Google Gemini API 이용약관에 따라 사용자(키 발급자)에게 있습니다. 다만 결과물이 기존 저작물과 유사할 수 있어 업로드 전 확인을 권장합니다.

업로드 음원의 저작권은?

본인 제작 또는 정당하게 라이선스 확보한 음원만 업로드 가능. 저작권 침해 책임은 업로드한 사용자에게 있습니다.

해상도는 어떻게 되나요?

디폴트는 1080p Full HD (1920×1080) 입니다. 프로/프리미엄 등급은 1440p (2K) 또는 4K UHD 도 선택 가능합니다. CPU 기반 libx264 인코딩, GPU 가속(NVENC/AMF)은 향후 업데이트 예정.

MacOS / Linux 도 되나요?

현재 Windows 10/11 64bit 전용입니다. macOS / Linux / 32bit Windows / 모바일은 미지원. 사양 미충족 환경에서 발생하는 동작 불량은 환불 사유로 인정되지 않습니다.

환불 정책은?

와디즈 펀딩은 와디즈 환불 정책을, 공식 판매 채널은 이용약관을 따릅니다. 라이선스 미활성 시 14일 이내 전액 환불 가능. 자세한 사항은 이용약관 참고.
